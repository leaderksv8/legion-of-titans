ГРОМАДСЬКА ОРГАНІЗАЦІЯ «ВЕТЕРАНИ ВІЙНИ З ОБМЕЖЕНИМИ МОЖЛИВОСТЯМИ БУЧАНСЬКОГО РАЙОНУ «ЛЕГІОН ТИТАНІВ»

Ми об'єднуємо ветеранів війни з обмеженими можливостями Бучанського району для взаємодопомоги, адаптації до мирного життя та розвитку. Наша місія - забезпечити підтримку, реабілітацію та нові можливості для тих, хто захищав нашу країну.
